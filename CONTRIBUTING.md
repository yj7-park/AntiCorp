# 기여 가이드

AntiCorp 프로젝트에 기여해주셔서 감사합니다!

## 이슈 제기

버그를 발견하거나 새로운 기능을 제안하고 싶으시다면 GitHub Issue를 생성해주세요.

## Pull Request

1. Fork the repository
2. Create your feature branch (`git checkout -b feature/amazing-feature`)
3. Commit your changes (`git commit -m 'feat: add amazing feature'`)
4. Push to the branch (`git push origin feature/amazing-feature`)
5. Open a Pull Request

## Commit 메시지 규칙

다음 형식을 따라주세요:

- `feat:` 새로운 기능
- `fix:` 버그 수정
- `docs:` 문서 수정
- `style:` 코드 포맷팅
- `refactor:` 코드 리팩토링
- `test:` 테스트 추가
- `chore:` 빌드 설정 등

## 코드 스타일

- C#: Microsoft 코딩 컨벤션 준수
- PowerShell: 가독성 우선, 명확한 변수명 사용
- Markdown: 간결하고 명확한 문서 작성

## 테스트
